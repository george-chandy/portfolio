Project 1

Atom IAS is a backend system for a user-centric educational platform built using Node.js and fully integrated with Firebase services. The platform manages user authentication, data storage, and provides a scalable backend infrastructure with minimal server management. This version leverages Firebase Authentication for secure, easy-to-manage user sign-ins and Firebase Firestore as the primary NoSQL database for storing user data and application content. The project is deployed and tested via Firebase Hosting, with a simple admin dashboard for real-time data monitoring and user management.

Project Overview

Atom IAS is a backend project built with Node.js that focuses on a secure and scalable user authentication system using Firebase Authentication. It features user registration, login, token-based authentication, and uses Firebase Firestore as the primary database.

I used Firebase Hosting for fast deployment and testing, and designed a basic admin dashboard to manage users and platform data. The project follows a clean, modular structure separating routes, services, and controllers. It is designed to be scalable and easily maintainable with Firebase-managed security and protected API endpoints.
