Project Overview

The Book Review App is a scalable, multitenant web application developed using FastAPI and PostgreSQL, enabling multiple companies or user groups to manage their book collections, categories, and reviews within isolated tenant environments.

Auth0 authentication ensures secure, external user management across tenants with role-based access. Users can create books, publish or unpublish them, manage authors and genres, and write detailed reviews. The application is designed with SQLAlchemy, Alembic, Pydantic, and is optimized for asynchronous, high-performance request handling using AsyncIO.

Future upgrades include AI-based personalized book recommendations, referral systems with coin-based rewards, session-based and blacklist-based logout features, and asynchronous background task management.
